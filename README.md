# Library Management System

A basic Library Management System developed using HTML, CSS, Java/Python, and MySQL to manage books in a library.

## Features
- Add new books
- View available books
- Store book details
- Manage library records

## Technologies Used
- HTML
- CSS
- Java / Python
- MySQL

## Project Files
- index.html
- style.css
- database.sql
- app.py / Main.java

## Purpose
This project helps librarians manage book details in a simple and efficient way.
